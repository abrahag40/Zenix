## 0. Principio editorial fundacional (LEER ANTES DE REDACTAR)

**Este curso NO es sobre cómo usar el PMS Zenix ni su módulo de housekeeping.** Es sobre **mejores prácticas de housekeeping hotelero alineadas a estándares globales** (AHLEI CHHE, CDC HAI, OSHA, EPA, Distintivo H cross). El objetivo es **agregar valor al hotel a través de su personal de limpieza** — usando estándares portables que el staff conserva incluso si cambia de hotel o de PMS.

**PROHIBIDO usar terminología Zenix interna:** sin PENDING/READY/IN_PROGRESS/DONE/VERIFIED, sin batchCheckout, sin task:ready, sin "2-phase checkout Zenix". Cuando se hable de estados de habitación o flujos de limpieza, se usan **conceptos genéricos universales de la industria** (vacante limpia / vacante sucia / room status codes universales, inspección de calidad estándar).

El "Curso Zenix Sistema" (cómo operar el módulo de housekeeping de Zenix por rol) es producto **separado**, gratuito vía Activate wizard, producido en fase posterior.

**Disclaimer obligatorio en cada lesson:** *"Curso ALINEADO al estándar AHLEI CHHE. NO emite certificación AHLEI oficial — el CHHE requiere examen pagado en AHLEI + prerrequisitos (2+ años experiencia managerial + 30h training). El comprobante Zenix Learning es interno alineado al estándar citado."*

### Módulo 1 — Introducción a operaciones de housekeeping (2h · 3 lessons)
- **L1.1** El error común: "limpiar es solo pasar el trapo" *(20 min)*
- **L1.2** Estructura del departamento + room status codes universales + MPI (Made Per hour) *(50 min)*
- **L1.3** La relación housekeeping ↔ front office ↔ mantenimiento *(50 min)*

### Módulo 2 — Planning, forecasting y productividad (3h · 3 lessons)
- **L2.1** El error común: asignar cuartos sin forecast *(30 min)*
- **L2.2** Forecast ocupacional + par stock + par level de lencería *(75 min)*
- **L2.3** Ratio recamarista/cuartos (AHLEI 14-16) + productividad + MPI *(75 min)*

### Módulo 3 — Químicos de limpieza + SDS + seguridad química (3h · 3 lessons)
- **L3.1** El error común: mezclar químicos (cloro+amoniaco = cloramina) *(30 min)*
- **L3.2** pH + tipos de químicos + código de colores + dilución correcta *(75 min)*
- **L3.3** SDS/MSDS 16 secciones + EPP + EPA-registered disinfectants *(75 min)*

### Módulo 4 — SOPs de limpieza por área (4h · 4 lessons)
- **L4.1** El error común: limpiar sin secuencia (recontaminar) *(40 min)*
- **L4.2** Bath strip + bed making (7 capas) + dusting top-down *(60 min)*
- **L4.3** Vacuuming back-out + terminal cleaning post-checkout *(60 min)*
- **L4.4** Deep cleaning + frecuencias + áreas comunes *(60 min)*

### Módulo 5 — Lencería y lavandería (3h · 3 lessons)
- **L5.1** El error común: lavar sangre con agua caliente (coagula la mancha) *(30 min)*
- **L5.2** Par stock lencería (PAR3) + manejo de manchas + temperaturas CDC *(75 min)*
- **L5.3** Proceso de lavandería + biohazard linens + EPP *(75 min)*

### Módulo 6 — Sustainability y amenities (2h · 2 lessons)
- **L6.1** El error común: reabastecer sin control (desperdicio + costo) *(30 min)*
- **L6.2** Programas opt-out reuso lencería + amenities + water/energy benchmarks *(90 min)*

### Módulo 7 — Bioseguridad + cross con Distintivo H (3h · 3 lessons)
- **L7.1** El error común: limpiar biohazard sin EPP *(40 min)*
- **L7.2** Protocolo biohazard (sangre, vómito) + EPA List K + doble bagging *(60 min)*
- **L7.3** Cross-contamination + áreas F&B compartidas + post-pandemic protocols *(80 min)*

### Módulo 8 — Maintenance liaison + Lost & Found (2h · 2 lessons)
- **L8.1** El error común: no reportar daños detectados durante la limpieza *(30 min)*
- **L8.2** Coordinación de tickets de mantenimiento + protocolo Lost & Found *(90 min)*

### Módulo 9 — Quality assurance + inspections (3h · 3 lessons + 🎮 Game F capstone)
> **Capstone (Fase 1.5):** 🎮 Game F *"Room Inspector"* — hidden-object educativo de inspección de habitación. Ver [doc 23 §2](../../23-games-interactive-learning-strategy.md).
- **L9.1** El error común: marcar habitación lista sin verificar *(30 min)*
- **L9.2** Room inspection score sheets + verificación de calidad *(60 min)*
- **L9.3** GSS (Guest Satisfaction Score) + mystery shopper + mejora continua *(90 min)*

### Módulo 10 — Seguridad del personal + ergonomía (3h · 2 lessons)
- **L10.1** El error común: cargar mal el carro (lesión lumbar) *(40 min)*
- **L10.2** Ergonomía + OSHA housekeeping safety + manejo seguro de químicos *(110 min)*

## 3. Criterios de evaluación

### 3.1 Examen final
- **50 preguntas multiple choice** (selección aleatoria de banco ≥250, ratio 5×)
- **1 práctica de inspección:** identificar errores en una habitación simulada (con foto-evidencia) — o Game F "Room Inspector" cuando esté disponible (Fase 1.5)
- **% aprobación:** 75% (el CHHE oficial usa score bajo NDA, típicamente 70%)
- **Re-takes:** 2 intentos con 48h espera; tercero requiere refresher de 4h

### 3.2 Module Quizzes
- Cada módulo cierra con 8-12 preguntas, aprobación ≥75%
- Drawn aleatoriamente del banco con Quiz Randomization Standard (doc 21 §1.1.bis)

### 3.3 Evidencia conservada
- Fotos antes/después de la práctica de inspección (data sintética)
- Score breakdown por módulo
- Registro de finalización para acreditar capacitación

## 6. Errores comunes universales que el curso DEBE corregir

> Cada uno se incorpora como "El error común" del hook de la lesson correspondiente.

1. **"Limpiar es solo pasar el trapo"** → M1 (housekeeping es ciencia de higiene + gestión + seguridad)
2. **"Asignar cuartos sin forecast"** → M2 (planning previene sobrecarga + descoordinación)
3. **"Mezclar químicos para limpiar más fuerte"** → M3 (cloramina = gas tóxico, cruza Distintivo H)
4. **"Limpiar sin secuencia"** → M4 (top-down + clean-to-dirty previene recontaminar)
5. **"Lavar sangre con agua caliente"** → M5 (coagula la proteína = mancha permanente)
6. **"Reabastecer sin control"** → M6 (desperdicio + costo descontrolado)
7. **"Limpiar biohazard sin EPP"** → M7 (exposición a patógenos = riesgo de salud)
8. **"No reportar daños detectados"** → M8 (cargo sorpresa al siguiente huésped + dispute)
9. **"Marcar cuarto listo sin verificar"** → M9 (caja fuerte, objetos olvidados, calidad)
10. **"Cargar mal el carro"** → M10 (lesión lumbar = la lesión laboral #1 en housekeeping)
